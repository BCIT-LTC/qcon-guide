# True/False (TF)

Correct answers are specified *inline* with an asterisk (*****).

!!! note

    **Only use the words `True` and `False`** as answer options or the question becomes a Multiple Choice type.

## Basic example

=== "Text"

        1. Canada Day falls on the first day of July.
            a. *True
            b. False

=== "Output"
